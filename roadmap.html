Architecture Layers

daub-classless.css Plain HTML looks good. Zero classes. Drop in one stylesheet and semantic markup gets considered aesthetics. Shipped
daub.css + daub.js 76 class-based components, 20 theme families with 40 variants. Tactile surfaces, letterpress type, real textures. Shipped
JSON-Render Spec AI outputs JSON, DAUB renders live UI. Working in the Playground — prompt to interface in seconds. Shipped
React / JSX Wrappers Typed components wrapping DAUB classes. <DaubCard>, <DaubButton> — full TypeScript, IDE autocomplete. Planned
Intent Engine Natural language in, UI spec out, live render. "Track my expenses" becomes a dashboard with charts and input forms. Future
Agent Runtime Multi-agent system orchestrates APIs + UI. Agents fetch data, decide layout, handle actions. UI is the surface. Future
Daub Place The endgame. You think it, it happens. Minimal interaction on device — ambient intelligence that anticipates and builds what you need before you finish asking. Vision

Release Timeline

Where we've been and where we're going.

v3.0
Classless Foundation

Classless CSS layer — plain HTML looks good with zero classes. ROADMAP published. Expanded AI docs with llms-compact.txt for token-efficient contexts.

v3.1
React Wrappers

Typed JSX components wrapping DAUB's CSS classes. Full TypeScript support, proper prop validation, IDE autocomplete. Framework-native without losing the zero-build soul.

v3.2
Streaming Render

Progressive UI construction during LLM streaming. Partial JSON arrives, partial UI appears. Components materialize as the model thinks — no waiting for the full response.

v3.3
Layout Intelligence

AI picks the right layout based on data shape. List of restaurants? Grid of cards. Single article? Reader view. Comparison? Side-by-side. The data decides the form.

v3.4
Action Bindings

Components that do things. Buttons that POST. Forms that submit. Cards that link to real endpoints. Generated UI isn't decoration — it's functional.

v4.0
Intent Engine

"I need to track my expenses" becomes a spending dashboard. Context-aware generation — location, time, history inform the UI. Adaptive complexity that reveals depth as you engage.

v5.0
Agent Runtime + Daub Place

Multi-agent orchestration: one fetches data, another decides layout, a third handles actions. The device becomes the place where thought meets interface — ambient, anticipatory, always ready.

Intent to Interface

You say it. The system builds it. No apps, no navigation, no learning curve.

Pizza

Cards of nearby places with ratings, ETA, prices, and an order button. Location-aware, time-aware.

Track my spending

Categorized expense dashboard with charts, input form, and monthly trends. Connected to your bank.

Plan a trip to Tokyo

Itinerary builder with maps, hotels, flights, day-by-day schedule. Drag to reorder, tap to book.

Help me write a cover letter

Editor with AI suggestions, tone controls, company research sidebar, and export to PDF.

What should I cook tonight?

Recipes based on what's in your fridge, dietary preferences, and prep time. Tap to start step-by-step mode.

Show me how my garden grows

Plant timeline, watering schedule, sunlight tracker, photo journal. Seasonal advice and harvest predictions.

Daub Place

The endgame isn't an app. It's a place where thinking and doing are the same thing. Your device becomes an extension of intent — ambient, anticipatory, considered.

Think It

Express intent naturally — voice, text, gesture. No app switching, no menus, no friction.

Shape It

AI parses intent, fetches data, chooses layout. UX laws baked in — Fitts, Hick, Miller. No broken UI.

See It

Purpose-built interface materializes. Clean, contextual, accessible. DAUB's tactile aesthetic throughout.

Do It

Interact, decide, complete. The interface serves the moment, then dissolves. Next thought, next interface.
